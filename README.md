## Dataset Overview

This dataset was independently generated by the author using a custom-developed project implemented in **C#**. The primary goal is to simulate power distribution network signals within a **cyber-physical environment**, with a focus on cybersecurity in OT power dispatching networks.

The data generation process is based on practical experience in industrial control systems (ICS/OT) and software engineering, following the framework proposed in:

> **Resilient LLM-Based Multi-Agent Generator (R-LMAG)**

This dataset is intended for academic research in:

* Power grid cybersecurity
* Data science
* Machine learning and artificial intelligence
* Multi-agent systems
